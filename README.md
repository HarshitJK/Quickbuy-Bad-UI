# QuickBuy Demo

A simple static e-commerce landing and cart demo built with HTML, CSS, and JavaScript.

## Project Files

- `index.html` – Shop page layout with product listing, sort controls, and cookie/marketing overlays.
- `cart.html` – Shopping cart page with order summary, countdown timer, and checkout CTA.
- `style.css` – Shared styling for both pages, banners, cards, buttons, and layout.
- `script.js` – Front-end behavior for cookies, newsletter popup, toast messages, product rendering, and sorting.

## Features

- Product grid with add-to-cart buttons
- Cookie consent banner and preference modal
- Newsletter overlay popup
- Sorting controls on the shop page
- Shopping cart UI with order summary
- Toast notification system

## Usage

1. Open `index.html` in a browser to view the shop page.
2. Click `Cart` to open `cart.html`.
3. Interact with the cookie banner, newsletter popup, and product buttons.

## Notes

- This is a static demo and does not persist any real cart data.
- Sorting and cart actions are UI-driven only.
